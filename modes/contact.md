# contact.md — Outreach Message Mode
> Trigger: "write a message to...", "cold outreach", "referral message", /contact

## Indian Context Rules
- LinkedIn connection request: 300 character limit. Respect it.
- Indian recruiters respond to: specific, short, direct. Not essays.
- Referrals work — most big Indian companies pay referral bonuses. Employees ARE incentivized.
- Never open with "I hope this message finds you well."
- Confident but not arrogant.

## Template 1 — LinkedIn Connection Request (recruiter, 300 char limit)
```
Hi {Name}, I'm {Your Name} — IIT Hyderabad grad, 1.5yr strategy analyst at Recykal (Python + SQL + ETL).
Saw {Company}'s {role} opening — looks like a strong match. Would love to connect.
```

## Template 2 — Referral Request (employee at target company)
```
Hi {Name},

I came across your profile researching {Company} — you're in {team}, which is the area I'm hoping to join.

Background: {Name}, IIT Hyderabad, 1.5yr at Recykal as strategy analyst — built ETL pipelines, Metabase dashboards, LLM automation. Applying for {role}.

If you're open to it, I'd value a 10-minute chat — and if the profile fits, a referral would mean a lot.

{LinkedIn} | No pressure if not a good time.
```
Under 150 words. The ask is clear. The escape clause is respectful.

## Template 3 — Follow-Up After Applying (14+ days, no response)
```
Subject: Following up — {Role} application ({Your Name})

Hi {Name}, I applied for {role} on {date} and wanted to follow up briefly.
I remain interested — particularly in {1 specific thing about the product/team}.
Happy to share additional work samples if useful.

{Name} | {Phone} | {LinkedIn}
```
Send once. 7 more days of silence = move on.

## Template 4 — Post-Interview Thank You (send within 24 hours)
```
Hi {Name}, thanks for the time today — I enjoyed the conversation, especially the discussion about {specific thing they mentioned}. Looking forward to next steps.
{Name}
```
Do NOT: say "I think it went well." Do NOT: repeat qualifications. 3 sentences max.

## Template 5 — Cold Email to Hiring Manager (no active posting)
```
Subject: Data analyst background — open to conversations at {Company}

Hi {Name},

I've been following {Company}'s work on {specific product/initiative} — the problem you're solving with {X} is exactly what I want to work on.

Anoj Kunte, IIT Hyderabad, 1.5yr at Recykal. Day-to-day: Python ETL pipelines, Metabase dashboards, LLM automation. Targeting data/analytics roles.

If there's appetite for an early-career data hire, I'd love 15 minutes. No expectation — just a conversation.

{CV attached}
```
This works. Most people don't do this.
